HOW TO USE THIS AS YOUR ACTUAL ICON

1. Open this HTML file in a browser, right-click the large (220px) icon and save it — or use a tool like Figma, Canva, or Sketch to recreate it as a proper 1024×1024 PNG.

2. Save it as assets/images/icon.png and also as assets/images/adaptive-icon.png in your project.

3. Your app.json already points to these files. Just rebuild with EAS:
eas build --platform all --profile preview

The font used is Arial Black / Helvetica Neue Black — available on all systems. For the final version a designer could swap this for a custom typeface to make it even more distinctive.
